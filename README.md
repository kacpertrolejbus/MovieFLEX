# MovieFlex 🎬

**MovieFlex** is a modern web application built for movie lovers to explore, rate, and organize their favorite films. The app integrates with the **OMDb API** to provide comprehensive information about thousands of titles, offering a seamless experience for managing a personal cinema database.
---

## 🛠 Tech Stack

* **Frontend:** [.NET Blazor]
* **Data Source:** [OMDb API]
* **Backend:** [ASP .NET WEB API]
